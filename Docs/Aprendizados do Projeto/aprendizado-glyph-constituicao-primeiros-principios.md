# Aprendizado — constituição antes do caso

## Ocorrido
Iterações recentes usaram “Economia Invertida” como âncora de feedback. O agent tratou melhorias como ajustes daquele JSON, enquanto o user pedia **primeiros princípios do sistema** (qualquer tese).

## Problema gerado
Opções de próximo passo soaram confusas; a gramática parecia colada a lifestyle/negócio/produto em vez de leis do glifo.

## O que funcionou
Separar: (A) escrever constituição universal; (C) auditar os 3 modos contra ela. Skill aponta para a constituição. Arc passou a desenhar arestas com seta (verbo+direção). Demos radial/arc usam `papel_semantico`, não `forma` solta.

## Solução / Lição
Caso de teste ≠ regra. Feedback visual num exemplo deve generalizar para leis; depois bater radial/arc/venn nelas.
