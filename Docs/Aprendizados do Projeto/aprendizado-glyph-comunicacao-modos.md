# Aprendizado — glifo precisa comunicar, não só conectar

## Ocorrido
MVP radial estilo iii ficou visualmente bom, mas mudo: nós iguais + mesh sem labels não explicava a tese. Refs Alan Hong (`refs_AlanHong/`) mostram modelos com metáfora espacial, título, zonas e narrativa.

## Problema gerado
Usuário via topologia bonita sem entender o argumento.

## O que funcionou
Expandir para modos `radial+` (núcleo + labels de aresta + narrativa) e `arc` (jornada U + zonas visível/oculto). Skill escolhe o modo; JSON carrega título/subtítulo/problema/trabalho.

## Solução / Lição
Mesh completo é ruído. Aresta sem verbo não comunica. Posição no canvas (centro vs órbita; acima/abaixo da linha) deve carregar significado. Próximos modos (`layers`, `polarity`, …) só depois de validar estes dois.
