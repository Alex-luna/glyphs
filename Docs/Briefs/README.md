# Briefs

Resumos técnicos do repo — gerados pela skill [`repo-brief`](../../.agents/skills/repo-brief/SKILL.md).

## Convenção

- Arquivo: `brief-{dd-MM-aa-HH-mm}-{lens}-{slug}.md`
- Lentes: `rules` (regras de negócio), `stack`, `both`
- Sem secrets — redact antes de commitar

## Quem escreve

Você + agent (skill `repo-brief`, ex.: `/brief rules geral`).
